# AI Surveillance Retrieval System

## Features
- 🎥 Video upload and detection (YOLOv8)
- 📊 Event extraction and storage
- 🤖 Query system using RAG
- 🖥️ Frontend dashboard (React)

## Tech Stack
- Python, FastAPI
- YOLOv8
- RAG (LLM + embeddings)
- React.js

## How to Run
### Backend
pip install -r requirements.txt
python api.py

### Frontend
cd frontend
npm install
npm run dev
